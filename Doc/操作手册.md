## 一、系统简介

**企业 AI 知识助手** 是一套基于人工智能的知识库问答系统，核心能力是：

1. **智能问答** — 用户用自然语言提问，系统从企业知识库中检索相关文档，由 AI 生成准确回答，并标注引用来源
2. **知识库管理** — 管理员上传文档（PDF、Word、Excel 等）或导入网页内容，构建组织的专属知识库
3. **数据分析** — 上传 Excel/CSV 文件，自动完成数据质量检查
4. **合规检查** — 上传文档与知识库中的制度规范进行比对，生成合规报告

系统支持多组织隔离，不同公司的数据完全独立，互不可见。

## 二、账号注册与登录

### 2.1 登录

1. 打开系统网址，进入登录页面
2. 输入**用户名**和**密码**
3. 点击「登录」按钮
4. 登录成功后自动跳转至智能问答页面

### 2.2 注册

1. 在登录页面点击「去注册」链接
2. 填写以下信息：
   - **用户名**：3~32 位，支持字母、数字、下划线和中文
   - **密码**：至少 6 位
   - **确认密码**：再次输入密码
   - **邀请码**：向管理员获取（必填）
3. 点击「注册」按钮
4. 注册成功后自动登录

> 💡 **关于邀请码**：
> - 每个组织有**管理员邀请码**和**普通用户邀请码**两种
> - 使用管理员邀请码注册 → 获得**管理员**角色
> - 使用普通用户邀请码注册 → 获得**普通用户**角色
> - 邀请码由超级管理员或公司管理员生成和分发

## 四、智能问答

智能问答是系统最核心的功能，基于 RAG（检索增强生成）技术，从知识库中检索相关内容，由 AI 生成准确回答。

### 4.1 开始问答

1. 点击侧边栏「智能问答」进入问答页面
2. 在顶部**部门选择器**中选择要检索的部门知识库（默认为「公共知识库」）
3. 在底部输入框中输入问题
4. 点击发送按钮或按 Enter 键提交

### 4.2 查看回答

- AI 回答会显示在对话区域
- 回答下方的「参考来源」标签展示了引用了哪些文档的哪些片段
- 点击来源标签可查看引用的原文片段

### 4.3 多轮对话

- 系统会记住当前会话的对话历史，支持多轮追问
- 例如先问「公司年假制度是什么？」，再追问「那工作满10年呢？」，AI 能理解上下文

### 4.4 切换部门

- 问答过程中可随时切换部门选择器
- 切换后新问题将在新选择的知识库部门中检索
- 会话历史保持连续，不受切换影响

### 4.5 导出问答报告

- 在 AI 回答下方点击「导出 PDF」按钮
- 系统将问题、回答和引用来源导出为 PDF 文件下载

> 💡 **提示**：
> - 如果知识库中没有相关内容，AI 会告知「知识库中暂无相关资料」
> - 描述问题越具体，检索效果越好（例如「2024年销售部报销流程」比「报销」更好）
> - AI 只基于知识库内容回答，不会编造信息

## 六、数据文件校验

上传 Excel 或 CSV 文件，系统自动完成数据质量分析。

### 6.1 操作步骤

1. 点击侧边栏「数据文件校验」
2. 选择要参考的知识库部门（可多选，如「公共知识库 + 人事部」）
3. 将文件拖拽到上传区域，或点击选择文件
4. 可选：在「分析要求」输入框中填写特别关注点（如「重点关注缺失值」）
5. 点击「开始分析」

### 6.2 分析结果

系统返回以下分析内容：

| 项目 | 说明 |
|------|------|
| **概览** | 文件名、行数、列数、内存占用 |
| **列分析** | 每列的数据类型、非空比例、统计值（均值/最大/最小）、异常值、常见值分布 |
| **缺失值** | 缺失单元格数量、比例、涉及的列 |
| **重复行** | 重复行数量和比例 |
| **异常值** | 偏离均值 3 倍标准差的数值 |
| **AI 分析报告** | 结合知识库制度规范的自然语言分析报告 |

### 6.3 支持的文件格式

- `.xlsx`、`.xls`（Excel）
- `.csv`（CSV，自动识别 UTF-8/GBK/GB18030 编码）

## 七、文档合规检查

上传文档与知识库中的制度规范进行合规比对。

### 7.1 操作步骤

1. 点击侧边栏「文档合规检查」
2. 选择参考的知识库部门
3. 上传文档（PDF、Word、TXT 等）
4. 可选：填写补充说明
5. 点击「开始检查」

### 7.2 检查结果

系统从以下维度进行合规分析：

| 维度 | 说明 |
|------|------|
| **符合项** | 文档中符合相关制度的条款 |
| **不符合项** | 文档中与制度不一致的条款（引用制度原文） |
| **风险评估** | 每个不符合项的风险等级（高/中/低） |
| **总体评估** | 整体合规评分（优/良/中/差） |
| **改进建议** | 针对不符合项的具体修改建议 |
| **参考来源** | 知识库中命中的制度文档列表 |

### 7.3 支持的文件格式

- `.pdf`、`.docx`、`.doc`（文档）
- `.txt`、`.md`（文本）

## 八、知识库管理（管理员）

> 📌 本节功能仅**管理员**和**超级管理员**可用。

知识库管理包含两个页面：「公司知识库」和「部门知识库」。

### 8.1 公司知识库 vs 部门知识库

| | 公司知识库 | 部门知识库 |
|---|----------|----------|
| **访问** | 所有用户 | 仅分配了该部门的用户 |
| **部门名** | general（公共知识库） | hr、tech、finance 等 |
| **用途** | 全员通用制度 | 部门专属文档 |

### 8.2 上传文档

1. 进入「公司知识库」或「部门知识库」页面
2. 选择目标部门
3. 点击「上传文档」按钮
4. 选择一个或多个文件（支持多选）
5. 等待上传完成

> ⚠️ **上传后必须重建知识库**（见 8.4），文档内容才能被智能问答检索到。

### 8.3 导入网页

1. 点击「导入网页」按钮
2. 输入网页 URL（仅支持 http/https 协议）
3. 点击「导入」

系统会自动抓取网页内容，提取正文后写入知识库。支持动态页面（SPA）的智能渲染。

> 💡 URL 导入是增量操作，不需要重建知识库，导入后立即可检索。

### 8.4 重建知识库

上传新文档后，需要重建知识库才能让 AI 检索到新内容。

1. 在知识库页面点击「重建知识库」按钮
2. 系统会重新读取所有文档，切分文本，生成向量索引
3. 重建过程中原有的 URL 导入内容会自动保留

> ⚠️ 重建可能需要几分钟（取决于文档数量和大小），期间该部门的知识库检索可能暂时不可用。

### 8.5 重建单个文件

如果只修改了一个文件，无需全量重建：

1. 在文件列表中找到目标文件
2. 点击「重建向量」按钮
3. 系统仅重新处理该文件，不影响其他文档

### 8.6 删除文档

1. 在文件列表中找到要删除的文档
2. 点击「删除」按钮
3. 确认删除

> 删除文档会同时移除文件和对应的向量数据。

### 8.7 删除向量（保留文件）

如果某个文件内容有问题，想暂时从检索结果中移除但保留源文件：

1. 点击「删除向量」按钮
2. 文件保留在磁盘上，但其内容不再被检索到
3. 后续可通过「重建向量」重新加入检索

### 8.8 预览文档内容

点击文档列表中的「预览」按钮，可查看文档在向量库中的切分内容，用于检查提取和切分效果。

### 8.9 修改文档标题

对于 URL 导入的文档，可以修改显示标题：

1. 点击文档的「编辑」按钮
2. 输入新的标题
3. 确认保存

## 九、知识库导出（管理员）

### 9.1 导出为 ZIP

1. 进入「知识库导出」页面
2. 选择部门
3. 选择导出范围：

## 十、用户管理（管理员）

### 10.1 查看用户列表

进入「用户管理」页面，显示当前组织下所有用户的信息：

| 字段 | 说明 |
|------|------|
| 用户名 | 登录账号 |
| 角色 | super_admin / admin / user |
| 状态 | active（正常）/ disabled（禁用） |
| 所属组织 | 用户所在的组织 |
| 部门 | 用户可访问的部门列表 |

### 10.2 创建用户

1. 点击「添加用户」按钮
2. 填写用户名、密码、选择角色
3. 新用户自动归属当前管理员所在的组织

> 💡 只有超级管理员才能创建 super_admin 角色的用户。

### 10.3 修改用户角色（仅超管）

1. 在用户列表中点击「修改角色」
2. 选择新角色（super_admin / admin / user）

### 10.4 修改用户所属组织（仅超管）

1. 在用户列表中点击「修改组织」
2. 选择目标组织

> ⚠️ 切换组织后，用户的部门分配会被清空，需重新分配。

### 10.5 启用/禁用用户

1. 在用户列表中点击「启用/禁用」按钮
2. 禁用后该用户无法登录，但数据保留

> ⚠️ 不能禁用超级管理员账号（除非你也是超管）。

### 10.6 分配用户部门

1. 在用户列表中点击「分配部门」
2. 勾选用户可访问的部门
3. 确认保存

> 💡 「公共知识库」(general) 默认所有人可访问，无需额外分配。

## 十五、组织管理（超级管理员）

> 📌 本节功能仅**超级管理员**可用。

### 15.1 查看组织列表

进入「组织管理」页面，显示系统中所有组织，包含：

- 组织编码、名称、状态
- 管理员邀请码、用户邀请码
- 用户数量

### 15.2 创建组织

1. 点击「添加组织」按钮
2. 填写以下信息：
   - **组织编码**（code）：全局唯一标识，如 `acme`、`company-a`
   - **显示名称**：如「某某科技有限公司」
   - **描述**：组织说明（可选）
3. 创建成功后系统自动：
   - 生成管理员邀请码和用户邀请码

### 15.3 编辑组织

修改组织的显示名称、描述或状态。

### 15.4 禁用组织

1. 点击「禁用」按钮
2. 该组织下所有用户将无法登录

> ⚠️ **默认组织**不可禁用。禁用是软操作，数据保留，可随时重新启用。

### 15.5 重新生成管理员邀请码

1. 点击「重新生成邀请码」
2. 系统生成新的管理员邀请码

### 15.6 切换组织视角

超级管理员可以在顶栏的**组织切换器**中切换当前查看的组织：

- 选择特定组织 → 只看该组织的数据
- 不选择 → 查看全局数据

## 附录 A：常见问题 FAQ

### Q1：注册时提示「邀请码无效」怎么办？

邀请码由管理员在「公司设置」或「组织管理」中生成。请联系管理员获取正确的邀请码。

### Q2：上传了文档但搜索不到内容？

上传文档后需要**重建知识库**（在知识库页面点击「重建知识库」按钮），文档内容才能被检索到。URL 导入的内容不需要重建，导入后立即可检索。

### Q3：AI 回答说「知识库中暂无相关资料」？

可能的原因：
1. 该部门的文档尚未上传或未重建知识库
2. 提问方式不够明确，尝试换种表述或使用更具体的关键词
3. 相关文档的相似度低于阈值

### Q4：忘记密码怎么办？

系统暂不支持自助重置密码，请联系管理员重置。

### Q5：为什么有些菜单看不到？

不同角色拥有不同的功能权限，请参考[角色权限对照表](#附录-c角色权限对照表)。

### Q6：导入网页时提示「网页抓取失败」？

可能的原因：
1. 网页需要登录才能访问
2. 网页地址为内网地址（系统禁止访问内网）
3. 网页服务器响应超时
4. 网页内容为空或无法提取正文

### Q7：数据分析/合规检查上传文件后报错？

请确认文件格式是否符合要求：
- 数据分析仅支持 `.xlsx`、`.xls`、`.csv`
- 合规检查仅支持 `.pdf`、`.docx`、`.doc`、`.txt`、`.md`

### Q8：超级管理员和普通管理员有什么区别？

- **超级管理员**：平台级权限，可管理所有组织、创建/禁用组织、修改用户角色和组织归属
- **普通管理员**：组织级权限，只能管理自己所属组织的用户和知识库

## 附录 B：支持的上传文件格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PDF | `.pdf` | 文字型 PDF（扫描版需先 OCR 处理） |
| Word | `.docx`, `.doc` | Microsoft Word 文档 |
| 文本 | `.txt` | 纯文本文件（自动检测编码） |
| Markdown | `.md` | 支持 Markdown 格式（按标题语义分块） |
| Excel | `.xlsx`, `.xls` | Microsoft Excel 电子表格 |

> 单文件大小限制：10MB（由管理员配置）

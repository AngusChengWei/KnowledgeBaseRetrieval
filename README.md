# 知识库问答助手 (KnowledgeBaseRetrieval)

一个基于 RAG 技术的企业级 AI 知识库问答系统，支持多格式文档上传、向量化存储与智能检索，帮助团队快速沉淀和复用内部知识。
## ✨ 功能特性

- 📚 支持 PDF/Word/Markdown/HTML 等多种格式文档解析
- ⚡ 基于向量数据库的高效语义检索
- 🤖 对接主流大模型，提供自然语言问答
- 🔒 权限控制与用户管理
- 📊 问答日志与使用统计

- ## 🚀 快速开始

### 环境要求
- Node.js 18+ / Python 3.10+
- 向量数据库（如 Milvus/FAISS）
- OpenAI/通义千问 API Key
